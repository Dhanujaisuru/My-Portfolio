# Dhanuja Isuru Portfolio

A modern, responsive personal portfolio website built with React and Tailwind CSS.  
This project showcases my profile, skills, projects, experience, and contact information with a clean dark-themed interface and polished hero section.

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
- CSS Variables
- Component-based architecture
